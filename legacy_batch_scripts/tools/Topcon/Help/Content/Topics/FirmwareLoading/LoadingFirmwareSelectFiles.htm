Selecting Firmware Files

1. Specify the full path to firmware files for the device by clicking the open folder icon or enter the full path manually.
If you are updating power board firmware or do not use a serial port connection, you should specify only the path to the Loader file (it usually has an .ldr extension). In other cases you should specify full paths to both Loader and Image files (usually with an .ldp extension).

To update the firmware through a connection other than serial one, the TRU application first needs to copy the Image (*. ldp or *.tfi file) file to an internal memory card of the receiver. If the receiver does not have the internal memory card or there is not enough free memory on the card, firmware updating process will fail.

2. Click Next to begin uploading firmware to the device.
