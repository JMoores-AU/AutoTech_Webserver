R2 Lite UHF Modem

This section describes specific parameters and functions for the R2 Lite UHF modem.

There is more powerful version of Digital UHF modem called TRL-35. It is identical to the Digital UHF, but supports different power values (up to 32 watts). TRL-35 has no cell modem.

General Information

The General tab shows the modem model, and the product identification information.

Cell modem – click the button to view information on the cell module.

Modem Settings

The Settings tab allows setting values for parameters of the connected modem.

The Settings tab displays general communication parameters, and parameters specific for the modem. To change any editable parameter, click on the Value cell of the highlighted parameter and select a desired value from the drop-down list appeared.

• General parameters:
  • Mode – allows to select Radio or Cell mode to use for communication (the UHF or GSM/CDMA band, respectively).
  • Baud rate, baud – allows to select a baud rate for the modem’s serial port.
  • Flow Control – allows to select hardware flow control.
• Radio parameters:
  • Spacing, kHz – allows to select the channel spacing.
  • RX Frequency, MHz – assigns an operating channel to transmit data. Select the desired frequency from the list.
  • TX Frequency, MHz – assigns an operating channel to receive data. Select the desired frequency from the list.
  • Modulation – selects a modulation type for the radio modem.
  • Protocol – sets the protocol for data receiving/transmitting.
  • Power, mW – sets the transmission power for the base radio.
  • Scrambler – when enabled, the data is encrypted to protect against unauthorized access to transmitted information.
  • FEC – (Forward Error Correction) when enabled, activates the operation to control errors in data transmission over unreliable or noisy communication channels.
  • Call Sign – a unique designation for a transmitter modem.
• Satel parameters:
  • Satel Mode – the Receiver / Transmitter mode is always enabled.
  • Satel FEC – (Forward Error Correction) when enabled, activates the operation to control errors in data transmission over unreliable or noisy communication channels.
  • Satel FCS Net ID, hex – a string of up to eight characters used to prevent the reception of the messages coming from an external modem's network. The radio modems operating in the same network must have the same Network ID. Only the messages that have a matching Network ID can be received.
  • Satel TX Addressing – specifies TX address mode, (On or Off).
  • Satel TX 1 Address, hex – configures the transmission primary address.
  • Satel TX 2 Address, hex – configures the transmission secondary address.
  • Satel RX Addressing – specifies RX address mode, (On or Off).
  • Satel RX 1 Address, hex – configures the receiving primary address.
  • Satel RX 2 Address, hex – configures the receiving secondary address.

Modem Functions

The Functions tab allows invoking modem's functions.

The following functions are available for executing in this modem:

• FactoryReset – resets all the parameters to their default values.
• UploadFile – used to upload Country Configuration (CCF) or Dealer Configuration (DCF) files.

 

Be aware that the R2 Lite UHF modem supports a wide UHF frequency range of between 400-470 MHz, and is delivered to you in a default factory configuration. The default factory configuration allows to support operation in 400-470 MHz using CCF configuration file.

Before operating a receiver equipped with a R2 Lite UHF radio, you must upload the frequencies needed for radio communication in your specific region. UHF frequencies in your region may be licensed or unlicensed, but are always subject to some level of restriction within the overall 400-470 MHz range.

Your dealer should load the DCF file before giving you the receiver integrated R2 Lite UHF radio. If your receiver is not loaded with UHF frequencies specific to your region, contact your local dealer and request that he supplies you with a DCF file containing licensed UHF frequencies and channel spacings.

Once you receive the DCF file from the dealer, you must have the latest release version of the TRU Application and do the following:

1. Install the Application on your Mobile or PC device.
2. Copy the DCF file provided by your dealer and paste it in hard drive of Mobile or PC device.
3. Run the TRU Application on your mobile or PC device.
4. Connect to the internal radio modem of the receiver, using the Modem Managing mode of the TRU Application.
5. Click the Functions tab, select the UploadFile in the Function field.
6. In the File property of the Input Parameters, select the DCF file saved on a local drive and select DCF as the type of file.
7. Click the Execute button to load the radio frequencies into the radio.
8. Open the Settings tab to ensure that all frequencies are correctly loaded and appear available for use.

 

A valid DCF configuration is mandatory before using the product.

Failure to understand and apply this procedure may result in non-operation of the R2 Lite UHF radio, or breach of local regulatory laws that govern radio communications.
