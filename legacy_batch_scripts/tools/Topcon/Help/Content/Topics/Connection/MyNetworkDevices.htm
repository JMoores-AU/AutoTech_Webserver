My Network Devices

You can add, edit, and delete your network receivers in the My Network Devices window.

• To add a receiver, right-click (PC) or tap (Mobile) anywhere in the window and select Add. Enter the network settings in the Network Connection window.
• To edit network settings of your receiver, right-click (PC) or tap (Mobile) the receiver you want to edit and select Edit. Enter the network settings in the Network Connection window.
• To delete a receiver, right-click (PC) or tap (Mobile) the receiver you want to delete and select Delete. Confirm whether or not you want to delete the receiver.
