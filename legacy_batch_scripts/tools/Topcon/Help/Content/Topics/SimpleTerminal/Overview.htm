Overview

In Simple Terminal mode, the Topcon Receiver Utility (TRU) allows a user to send commands through the selected port.

In Simple Terminal mode, the Topcon Receiver Utility (TRU) does not perform the following tasks:

• Detecting a device
• Adjusting port settings
• Sending any data to the port

This mode is intended for advanced users. Everything is decided by the user in this pure manual mode.

Simple Terminal mode can be used, for example, if you want to connect to a device that the Topcon Receiver Utility (TRU) does not support.

To make the tool area active, first establish a port connection on the controller using the Connect option from the Device menu.

In the Main screen the following icon is available.

The Simple Terminal icon allows you to communicate with any device in manual mode. You can record data to a file from a connected device.
