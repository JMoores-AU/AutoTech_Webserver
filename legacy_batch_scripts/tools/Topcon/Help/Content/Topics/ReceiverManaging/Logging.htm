Logging

The Logging tab allows logging real-time measurements into files and saving them in the computer or controller.

  • File Path – full path to the file in which data will be logged. Use the button to navigate.
  • Check box Multiple Files – when selected, the next file logging starts immediately after automatic completion of logging of the previous file. The parameters set for the one file will be used for logging of all the following files. Click the Settings button to configure naming scheme and duration of multiple files.
  • Elevation Mask - defines the minimum angle above the horizon, in degrees, for satellites from which the receiver will send data into the file.
  • Period - sets the data logging interval in seconds.
  • Check box Default - when selected, a default period will be used for each message of the raw file. In this case you will be unable to set a recording period manually.
  • Statistics – shows file logging statistics in progress after starting logging the file.
  • Start Logging button – press the button to start logging the file. After pressing, the button changes into Stop Logging.
  • Logging Messages button – opens the def<n> dialog to view edit a list of messages being logged into the file and logging parameters .
  • Site Parameters button – opens the Site Configuration dialog to apply antenna parameters for data logging purposes at the site.

If one or more message(s) in the list has a period less than the position or raw measurements update rate of the receiver, an error message will display. To log a file with such a period you have to change the update rates in the Adv tab.
