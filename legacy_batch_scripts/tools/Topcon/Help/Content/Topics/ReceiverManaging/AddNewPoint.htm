Add or Edit AutoSeed Point

Enter the ID and Name of the new point. Check mark the Auto box to enable averaging mode for auto detection of coordinates of the phase center.

Press the corresponding button to enter appropriate coordinates in the Geodetic or Cartesian coordinates depending on which coordinate system you select in the Setup window on the General tab. You can also use this dialog to edit the coordinates of existing points.
