HTTP

For the HiPer VR, HiPer HR and Net-G5receivers, the HTTP tab allows you to configure passwords for two levels of account types: user and admin. You can also specify TCP ports the web server will listen on for incoming http or https connections.

With an administrator-level account, you can view and change all receiver settings. Only one administrator account exists per receiver. The administrator can edit a password associated with the user account type.

With a user-level account, you can view and change all receiver settings with the following exceptions:

• cannot change credentials (password) of administrator account;
• cannot upload firmware or OAF files;
• cannot do a factory reset of the receiver;
• cannot change network settings;
• cannot change specific advanced settings.

There is also a guest-level account, which provides read-only permissions and do not allow any changes of receiver settings. Guest accounts are for demonstration purposes only with a limited number of settings that a guest user can view.

For the Net-G3A receiver, the HTTP tab allows you to configure credentials (Login and Password) that will be used for logging in through the web interface.

In the Ports settings, specify TCP ports the web server will listen on for incoming http or https connections.
