Events

The Events tab allows editing event triggers:

• Minimum Number of Tracking Satellites — the receiver sends notification, if the number of tracked satellites is lower than specified.
• Minimum Charge Level of Battery A, % — the receiver sends notification, if the level of Battery A charge is lower than specified.
• Minimum Charge Level of Battery B, % — the receiver sends notification, if the level of Battery B charge is lower than specified.
• Minimum Available Disk Space, MB— will send notification, if the amount of free disk space is lower than specified.
• Any OAF expires in less than, days — will send notification in specified amount of days before expiration of your Option Authorization File.
• Maximum Base Station Offset — will send notification, if the difference between the base station position (entered by a user) and the calculated base station position is more than specified.

Right-click (PC) or tap (Mobile) any line and select Edit in the context menu. The Event dialog is opened, where you can change the event threshold value. You can also select the period of the notifications (Notify Once or Notify Periodically), if the option is available for the event selected.
