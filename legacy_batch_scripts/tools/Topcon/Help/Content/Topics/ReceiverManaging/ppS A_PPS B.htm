PPS A / PPS B

In the PPS A and PPS B tabs you can configure Pulse Per Second (PPS) signal for output A and output B accordingly:

  • Enable check box — enables the receiver to generate PPS signals, as well as modify the settings related to the selected PPS signal.
  • Period, ms — determines the period of generating PPS signals via the corresponding output connector.
  • Offset (ns) — allows to input an offset between the scheduled PPS time and the actual generation time of the pulse edge in nanoseconds.
  • Generate Marked Pulses check box - when selected, the marked pulse will be created (width of the marked pulse is more than width of a normal pulse in 1.5 time)
    • Period, ms — specifies the period of the marked PPS signal in milliseconds;
  • Edge: — synchronizes the edge (rising or falling) of the PPS signal with the specified reference time.
  • Ref. Time — selects the reference time that the PPS signal will be synchronized with; it could be either GPS system time, GLONASS system time, UTCus (UTC USNO) system time, or UTCsu (UTC SU) system time.
  • Tied with Ref. Time check box — when selected, the receiver will synchronize the PPS signal with the chosen reference time. Otherwise, PPS will be synchronized with either the receiver's internal clock or with an external reference frequency applied to the receiver.
