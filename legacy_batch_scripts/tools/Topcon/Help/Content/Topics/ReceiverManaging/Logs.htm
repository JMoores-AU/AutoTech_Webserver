Logs

The Logs tab allows you to view and to specify some individual settings of a log file.

• Log – selects current file.

For the HiPer II and HiPer V receivers, you can configure only one current file – A.

For the HiPer SR, GR-5 and Net-G3A receiver, you can configure two current files – A and B.

For the HiPer VR, HiPer HR and Net-G5 receivers. , you can configure up to sixteen files – from A to P.

• File Name – specifies the name of the file, which will be created.
• File Size, B – shows the size of the current file.
• Current Logging Period, s – specifies the rate at which the receiver is recording the log files. You can use the following types of this parameter:
  • Variable – for each message of the global default message set, its own period is used.
  • a number – for each message of the global default message set, a period which you specify in the field, is used.
• Elevation Mask, ° – specifies the elevation masks threshold value for TPS messages to be output to the file.
• Prefix – specifies the prefix used as part of the name of the new file that the receiver creates.
• MINTER and Logging Period, s for HiPer II and HiPer V receivers – specifies the interval to be used for data logging after the MINTER FN key is pressed. It can be set to the values from 0 to 86400 seconds; the default is 1 second.
• MINTER and AFRM Logging Period, s for HiPer SR, HiPer VR, GR-5, Net-G3A, HiPer HR and Net-G5 receivers – specifies the interval to be used for data logging after the MINTER FN key is pressed. This parameter is also applied to AFRM mode. It can be set to the values from 0 to 86400 seconds; the default is 1 second.
• AFRM Rotation Period, s for HiPer SR, HiPer VR, GR-5, Net-G3A, HiPer HR and Net-G5 receivers – shows the duration time of each log file created in AFRM.
• Use for MINTER & AFRM for HiPer SR, HiPer VR, GR-5, Net-G3A, HiPer HR and Net-G5 receivers – shows using ("Yes") or ignoring ("No") of the specified log parameters after pressing the MINTER FN key and starting of AFRM.
• RINEX version for HiPer HR and Net-G5 receivers – shows either enabled ("RINEX_Version") or disabled ("None") conversion of the current log file to the RINEX file. You can select the desired RINEX version in the Log dialog (click Edit in the context menu to open the dialog).

The conversion of the log file to the RINEX file format is started immediately after the log file is closed.
• Message Set for HiPer VR, HiPer HR and Net-G5 receivers – shows the message set number being logged in the log file. You can select the desired message set in the Log dialog (click Edit in the context menu to open the dialog).

Right click on any place of the tab to open the following context menu:

• Edit – opens the Log dialog, where you can configure the log file settings.
• Start File Logging (Default Settings) – allows you to start a new log. After selecting the command, the receiver generates a file name automatically. The default message set’s schedule is used for each message. The Current Logging Period, s column displays in the Variable column.
• Start File Logging (Manual Settings) – opens the Start Logging dialog, where you can specify a file name and select a way to define a logging period for a file to be created.
• Stop File Logging – stops the currently logged file.
• Advanced MINTER Settings for HiPer SR, HiPer VR, GR-5, Net-G3A, HiPer HR and Net-G5 receivers – opens the Minter dialog, where you can edit advanced settings for the Minter.
• Default Message Set – opens the def <n> dialog, where you can view and edit a list of messages being logged into the file, and also parameters of the selected message.
• Site parameters - opens the Site configuration dialog, where you can edit the site and antenna parameters for data logging purposes.
• View – allows you to display / hide columns, related to described settings:
  • Current Log – displays / hides the File Name, File Size, B, Current Logging Period, s and Elevation Mask, ° columns.
  • MINTER Settings for HiPer II and HiPer V receivers – displays / hides the Elevation Mask, °, Prefixand MINTER Logging Period, s columns.
  • MINTER& AFRM Settings for HiPer SR, HiPer VR, GR-5, Net-G3A, HiPer HR and Net-G5 receivers – displays / hides the Elevation Mask, °, Prefix, MINTER and AFRM Logging Period, s, AFRM Rotation Period, s and Use for Minter & AFRM columns.
  • RiNEX Settings for HiPer HR and Net-G5 receivers – displays / hides the RINEX Version column.
