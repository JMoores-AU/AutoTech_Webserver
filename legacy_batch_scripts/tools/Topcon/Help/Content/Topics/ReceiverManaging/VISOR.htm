VISOR Parameters

The VISOR tab allows you to configure the MR-1 receiver / OEM-1 and B 210 board for 2-D attitude determination (heading and inclination).

Prior to configuring, make sure that the Activate Twin Antenna System check box on the Antenna tab is selected.
• Enable VISOR – if it is selected, the receiver computes 2-D attitude parameters. In this case the user can select the initialization type.
• Use L2 for VHD - if it is selected, the VHD engine uses L2 measurements to compute 2-D attitude parameters. This option is available for B210 OEM board only.

The following types of VISOR initializations are available:

  • Velocity Vector – initializes VISOR by moving the platform with rigidly mounted antennas.
  • Twin Antenna Rotation – initializes VISOR by rotating a platform with rigidly mounted antennas in a plane parallel to horizon.
  • Manual – enters initial heading and inclination with the GRIL command.
  • Auto – initializes using initial heading and inclination computed by the auxiliary RTK engine running in the background.
• Antenna Separation – specify the inter-antenna distance in Manual Value. The Computed Value field displays the automatically computed distance. Check the Save Computed Value setting to instruct the auxiliary RTK engine to compute and use the inter-antenna distance.
• Advanced – specify some advanced VISOR settings.

You can view computed 2-D attitude parameters on the Attitude tab.

For more information about VISOR configuration, limitations, and guidelines, see the OEM-1 Integrator's Guide, Enabling VISOR and Understanding Results.
