Slot <'N'>: Properties; Driver tab

The Drivertab displays information about the currently installed driver, and allows updating or uninstalling the driver.

 

The tab contains the following parameters:

• Driver Name – an internal name of the driver;
• Driver Version – the version of the driver;
• Driver Description – a user friendly name of the driver;
• Driver Details button – clicking the button opens the Driver Files Details dialog, where you can see a list of profiles and scripts which the driver consists of.
• Update Driver button –clicking the button opens first page of the Wizard, which helps you to update the driver or install a new one.
• Uninstall button – allows to erase the driver, its internal variables and the information about the modem from permanent NVRAM of the receiver.
