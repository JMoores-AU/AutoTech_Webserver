Tools

The Tools icon opens the Tools dialog box that allows you to reset the receiver and to clear the NVRAM.

• Reset Receiver — performs a hardware reset that functionally equivalent to power cycling the receiver. Resetting the receiver leaves all receiver parameters and files intact.
Typical use of this tool is to apply certain settings which require a hardware reset. After completing this operation, the receiver requires from a few seconds to a few minutes to begin tracking satellites and logging data.
• Factory Reset — resets all the receiver parameters to their default values and clears the receiver’s Non-Volatile Random Access Memory (NVRAM). NVRAM holds data required for satellite tracking, such as ephemeris data and position, and also keeps the current receiver’s settings, such as active antenna input, elevation masks, etc.
Factory Reset does not delete any files from the receiver and does not reset modem parameters.
Typical use is after loading a new firmware, and sometimes to eliminate communication or tracking problems. After performing this procedure, the receiver will require some time to collect new ephemerides and almanacs (up to 15 minutes).
• Reset Parameters — resets all the receiver parameters to their default values. This operation does not delete files, almanac and ephemerides from the receiver and does not reset modem parameters.
• Reset RTK — resets the RTK engine.
• Power Off — turns the receiver off. Before using this command, make sure you have physical access to your receiver for manual turn-on because once this command is sent, you cannot turn the receiver on via the Application.
• Sleep — puts the receiver into sleep mode. The receiver enters into sleep mode immediately after you click Yes on the confirmation prompt and stays in this mode until you turn on the receiver manually or the receiver detects any activity on one of the serial ports.
• Reset VISOR — resets the VISOR engine.
