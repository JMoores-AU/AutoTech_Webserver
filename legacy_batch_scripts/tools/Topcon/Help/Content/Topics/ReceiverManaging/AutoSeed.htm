Auto Seed

The Auto Seed feature allows to quickly setup and begin performing RTK operations without using an external interface to localize the base receiver. The user simply sets up the receiver over a ground mark and powers the receiver on. The base receiver automatically selects a new position gathered by autonomous averaging and saves it for further re-occupation. If the point has been previously occupied and the receiver position falls within proper tolerance, it will select a point from positions stored in memory.

• Enable Auto Seed- if it is selected, the Auto Seed functionality is activated.
• Position averaging interval, s- sets the interval for autonomous averaging.
• Check Distance check box - if it is selected, you can enter the distance tolerance.
• Maximum distance, m- tolerance for the distance from the previously occupied point.
• Enable averaging mode- when selected, enables averaging autonomous positions for an occupation point.
• Average only RTK Fixed solutions check box - if it is selected, the firmware will average the receiver coordinates, when solution type is RTK Fixed.
• Max 3D RMS, m - allows to enter the values of the final 3D RMS. When the values are achieved, the averaging operation will be stopped automatically (and a point with the average coordinates will be created). If during averaging of RTK Fixed solution such RMS value will be not achieved, the averaging operation will be stopped after achieving of average value, set in the Position averaging interval, s field.
• View the point list button - opens a list of previous occupations saved in the receiver memory.
