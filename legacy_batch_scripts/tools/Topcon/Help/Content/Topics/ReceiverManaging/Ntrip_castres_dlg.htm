Ntrip Casters

The Ntrip Casters dialog allows you specify for each caster individual parameters. The dialog contains three identical tabs:

• Accept Ntrip Request check box — when checked, the Caster is activated and ready to listen to the TCP Port for incoming clients for a mount point.
• TCP Port Number — specifies the nonzero TCP port which will be used by the NtripCaster.
• Mountpoints — shows the list of the mounpoints for the NtripCaster. This list can contain no more than 3 mounpoints.

Click the Add button to open Mountpoint dialog, where you can create a new mountpoint for the NtripCaster. You can not create more than three mountpoints.

Click the Edit button to open Mountpoint dialog, where you can edit parameters of the highlighted mountpoint.

Click the Delete button to delete the highlighted mountpoint.
