Start Logging

The Start Logging dialog allows you to start recording to the current log file with the following parameters:

• Current Log – the current file name.

In the File Name area you can select a way of creating of the file name:

  • Auto Generate – when selected, the receiver will assign a file name automatically.
  • Empty field– when selected, the receiver will use a name you enter in this field.

In the Logging Period, s area you can select a way of specifying the logging period:

• Default Message Set's Schedule – when selected, for each message of the global default message set, its own period will be used.
• Default Period for MINTER & AFRM – when selected, for each message of the global default message set, a period which you specify in the Logging Period field (Log dialog), will be used.
• Empty field– when selected, for each message of the global default message set, a period which you specify in this field will be used.
