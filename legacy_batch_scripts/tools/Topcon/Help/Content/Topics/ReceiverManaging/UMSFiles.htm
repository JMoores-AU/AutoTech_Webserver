UMS Files

The UMS Files tab displays the list of raw data files stored on the UMS device connected to the Net-G3A receiver.

Along with the file names, this list contains the file size in bytes, and date and time the files were recorded.

The UMS Files tab is also used to manage the files. The pop-up menu displays the following settings:

• Refresh – reloads the list of files.
• Delete – deletes the selected TPS files. A confirmation message displays.
• UMS Device Info – displays information about the UMS device connected to the receiver, including its name, firmware version, file system details, and more.
