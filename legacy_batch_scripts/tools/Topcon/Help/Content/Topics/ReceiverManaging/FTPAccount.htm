FTP Account

The dialog allows to configure an FTP server using the following settings:

• IP Address – IP address of the FTP server to which the receiver (FTP client) will connect.
• Login – the user name required by the FTP server for a client to connect. The default is anonymous. The user name is case sensitive and can contain up to 31 alphanumeric characters.
• Password – the password required by the FTP server for a client to connect. The default value depends on the receiver. The password is case sensitive and can contain up to 31 alphanumeric characters.

FTP Push and RINEX converter use these settings to transfer files to an FTP server.
