L Band

The L Band tab configures the L-Band receiver of the GNSS receiver:

• Service: — select the desired correction service from the list. If you select None, the L- band receiver will not get a correction data. Currently this service supports TopNET Global correction service.
• Satellite:— from the drop down list, you can select a satellite that will be used as a source of real-time corrections:
  • Auto — stands for automatic scan. In this mode, the receiver will automatically search for available satellite and stop the searching process as soon as the satellite is locked onto;
  • User Defined — allows you to lock onto the desired satellite by adjusting the corresponding tracking parameters manually;
  • AORE — stands for the AOR (east) satellite;
  • AORW — stands for the AOR (west) satellite;
  • IOR — stands for the IOR satellite;
  • POR — stands for the POR satellite;
  • 25E — stands for the 25 (east) satellite;
  • 143.5E — stands for the 143.5 (east) satellite;
  • 98W — stands for the 98 (west) satellite.
• Frequency: — shows the carrier frequency used by L-Band receiver to get the correction data from the selected satellite. You can edit this value for the User Defined selection in the Satellite drop down list.
• Symbol Rate: — shows the symbol rate that corresponds to the satellite you want to use. You can edit this value for the User Defined selection in the Satellite drop down list.
• Service ID, hex: — shows the service identification parameter. You can edit this value for the User Defined selection in the Satellite drop down list.
• Scrambler Vector , hex: — shows the service scrambler vector. You can edit this value for the User Defined selection in the Satellite drop down list.
