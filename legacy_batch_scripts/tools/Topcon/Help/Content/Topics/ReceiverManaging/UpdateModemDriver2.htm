Updating Modem Driver: Selecting Driver

This page allows you to select a driver from a list of all the drivers found in the file specified on the first page.

The Wizard does not actually detect modems. It uses information about the modems previously stored in the permanent NVRAM of the receiver. If your modem(s) have not been detected yet, it is recommended to detect them using the Scan for hardware changes menu item in the Modems dialog, or using the Detect Modems button of the Modem Properties dialog.

If there is only one driver, which is compatible with the hardware, the Application will highlight it. Otherwise you will have to select a driver manually.

To view all drivers included to the XML file, clear the Hide incompatible drivers checkbox.

To pass to the next page of the Wizard, click the Next button.
