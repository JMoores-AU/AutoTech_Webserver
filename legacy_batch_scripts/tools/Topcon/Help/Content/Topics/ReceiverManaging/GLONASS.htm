GLONASS

The GLONASS tab enables or disables the receiver to track and use GLONASS satellites.

• Track (FCN) – lists FCN's of all GLONASS satellites. Check mark corresponding boxes to allow the receiver to track selected GLONASS satellites. Use the pop-up menu options to enable or disable tracking of all satellites.
• Use for Positioning (FCN) – lists FCN’s of GLONASS satellites that you can use for positioning. Check mark corresponding boxes to allow the receiver to use selected GLONASS satellites. Use the pop-up menu options to enable or disable usage of all satellites.

The ability to track and use GLONASS depends on the receiver model, OAF, and firmware version you are running.

Right-click (PC) or tap (Mobile) anywhere in the dialog to bring up the following controls:

• Track all satellites – checks all checkboxes for all GLONASS satellites in the Track (FCN) panel. After that the GNSS receiver will track all GLONASS satellites.
• Don't track any satellites – unchecks all checkboxes for all GLONASS satellites in the Track (FCN) panel. After that the GNSS receiver will not track the GLONASS satellites and these satellites will not be used in the positioning.
• Use all satellites – checks all checkboxes for all GLONASS satellites in the Use for Positioning (FCN) panel. After that the GNSS receiver will use in the positioning all tracked GLONASS satellites.
• Don't use any satellites – unchecks all checkboxes for all GLONASS satellites in the Use for Positioning (FCN) panel. After that the GNSS receiver will not use the GLONASS satellites in the positioning.
