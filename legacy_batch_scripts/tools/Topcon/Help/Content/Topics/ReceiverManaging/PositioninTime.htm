Position in Time

The Position in Time tab displays the receiver position in real time.

Three charts on the screen represent 3D position in time: Latitude, Longitude, and Altitude. Each chart shows both absolute and relative values, and the time scale. The time scale varies from 1 to 5 hours.

You can apply a unique color to each solution type using the Status tab of the Setup dialog box.

Use the following buttons in the lower part of the screen to customize the graphic view.

Button Description
Zooms the plot in.
Zooms the plot out.
Clears all points from the screen.
