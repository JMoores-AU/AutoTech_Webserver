Backpack Tools

When TRU is running on Tesla or FC-2600 controller, the Device drop-down menu contains the Backpack Tools item.

The Backpack Tools item allows to open a dialog used to control power parameters of the Topcon GNSS receiver located on the rear panel of the controller. The dialog contains the following options:

• Current power state – shows the current power state of the internal GNSS receiver, either ON or OFF.
• Power ON/ Power OFF – turns on/off the internal GNSS receiver power.
• Reset the receiver – performs a hardware reset of the internal GNSS receiver. From a functional point of view, the hardware reset is equivalent to turning the power off and then back on.
• Turn power ON when connecting to the internal receiver – select this checkbox to allow the Application to turn ON the internal GNSS receiver, when connecting to it.
• Turn power OFF when disconnecting from the internal receiver – select this checkbox to allow the Application to turn OFF the internal GNSS receiver, when disconnecting from it.
• Turn power OFF on exit – select this checkbox to allow the Application to always turn OFF the internal GNSS receiver power on exit.
