Positioning

The Positioning tab allows changing positioning mode, and enabling solutions.

  • Positioning Mode – selects one of the following positioning modes for the receiver.
    • Standalone - the receiver will work in the single point positioning mode. Differential corrections from a base station are not available;
    • DGNSS - the receiver will work in the code differential mode;PP
    • RTK Float - the receiver will work in the RTK mode with float ambiguities or 2-D attitude determination in HD2 mode with float ambiguities;
    • RTK Fix - the receiver will work in the RTK mode with fixed ambiguities or 2-D attitude determination in HD2 mode with fixed ambiguities;
    • PPP - the receiver will work in PPP (TopNET Global ) mode with float ambiguities;
    • PPP and RTK - the receiver will work in PPP (TopNET Global ) mode with float ambiguities and possibility to use RTK with fixed ambiguities if correction data are available. Two engines work simultaneously and allow to have the best solution and seamless transition from RTK solution to PPP solution without convergence period.
  • Enable Solutions – selects one of the following solution types:
    • Standalone - specifies to output the current standalone position if a DGPS (or RTK) position is unavailable in the current epoch.
    • DGNSS - specifies to output the current code differential position if an RTK position is unavailable in the current epoch.
    • RTK Float - this position will be outputted always if an RTK Fixed position is unavailable in the current epoch.
    • RTK Fixed - this position will be outputted always if an RTK Fixed position is available in the current epoch.
  • Dion - selects mode in which the DION engine will operate. DION is an engine that allows you to compute smoothed and consistent position solutions. Instead of having instantaneous yet rather noisy positioning result, you can activate the DION engine to get more accurate and reliable solution.
    • Off - the DION engine is disabled.
    • Local- the DION engine provides precise positioning with respect to a starting point.
    • Smooth- the DION engine provides smoothed estimates of positions using refined delta positions from carrier phase processing.
    • HD2 Base - the DION engine provides exceptionally smooth and stable, but, often, shifted position estimates, required for implementation of Moving Base mode and for attitude determination in HD2 mode.

In DION Local, the Positioning Mode area is grayed out.
