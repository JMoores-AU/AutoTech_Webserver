Scatter Plot

The Scatter tab displays the receiver's trajectory/position in real time.

The scatter plot is updated each second. The current position is displayed by appropriate icon.

Each solution type is designated by a respective icon.

Icon Solution Type
Standalone, DION
DGNSS, OmniSTAR VBS
RTK Float, Variable Float
RTK Fixed, Variable Fixed, OmniSTAR HP, OmniSTAR XP

You can apply a unique color to each solution type using the Status tab of the Setup dialog box.

Use the following buttons in the lower part of the screen to customize the graphic view.

Button Description
Zooms the plot in
Zooms the plot out
Clears all points from the screen
Fits all points in the plot
Moves the center of the plot to the average of all points

It is possible to click, and drag the plot to change the center point. On a PC version the mouse wheel can be used to change the scale.
