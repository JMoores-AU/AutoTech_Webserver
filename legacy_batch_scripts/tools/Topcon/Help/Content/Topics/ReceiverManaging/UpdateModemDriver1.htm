Updating Modem Driver: Selecting Driver Source

This page allows you to select a driver source. When it is selected, the Application displays the file version.

The driver source is an XML file, which consists of one or more modem drivers. The file version increases when any changes to the file are made and it is not related to the version of the driver for your specific modem(s). The driver version is displayed on the second page of the Wizard.

Every Application installation includes a default driver library for all the officially supported equipment as of the release date of Application. If you have a newer library, you may use it instead of the default one. To select a new XML file, clear the Use by default check box.

To pass to the next page of the Wizard, click the Next button.
