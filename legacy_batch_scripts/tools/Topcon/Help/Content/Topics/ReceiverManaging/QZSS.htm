QZSS

The QZSS tab enables or disables the receiver to track and use Quasi-Zenith Satellite System (QZSS) satellites.

• Track (PRN) – lists PRN's of all QZSS satellites. Check mark corresponding boxes to allow the receiver to track selected QZSS satellites. Use the pop-up menu options to enable or disable tracking of all satellites.
• Use for Positioning (PRN) – lists PRN’s of QZSS satellites that you can use for positioning. Check mark corresponding boxes to allow the receiver to use selected QZSS satellites. Use the pop-up menu options to enable or disable usage of all satellites.

The ability to track and use QZSS depends on the receiver model, OAF, and firmware version you are running.

Right-click (PC) or tap (Mobile) anywhere in the dialog to bring up the following controls:

• Track all satellites – checks all checkboxes for all QZSS satellites in the Track (PRN) panel. After that the GNSS receiver will track all QZSS satellites.
• Don't track any satellites – unchecks all checkboxes for all QZSS satellites in the Track (PRN) panel. After that the GNSS receiver will not track the QZSS satellites and these satellites will not be used in the positioning.
• Use all satellites – checks all checkboxes for all QZSS satellites in the Use for Positioning (PRN) panel. After that the GNSS receiver will use in the positioning all tracked QZSS satellites.
• Don't use any satellites – unchecks all checkboxes for all QZSS satellites in the Use for Positioning (PRN) panel. After that the GNSS receiver will not use the QZSS satellites in the positioning.
