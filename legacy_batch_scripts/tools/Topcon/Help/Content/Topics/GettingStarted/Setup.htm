Setup

To change general parameters, select Device >Setup from the main menu. The Setup dialog box contains the following tabs: UI, Receiver, Status, and Modems.

User Interface (UI)

In the UI tab, you can manually set the UI scale ratio for changing the controls size. There are the following options:

  • Auto Scaling check box - if it is selected, the size of the application controls (icons, buttons and etc.) are automatically scaled to fit the size of the current PC / Controller screen. In this case the size of controls will be the same for the screens with the different resolutions.
  • Scaling Ratio - specifies the UI scale ratio to change the size of application controls. To zoom in controls, you need to enter the value more then 1, to zoom out the controls, you need to enter the value more then 0, but less than 1. This option is enabled, if the Auto Scaling check box is not selected.

If the application is installed on the Controller where the Table mode is available (e.g. FC-5000 with operation system Windows 10), the following check box is displayed:

  • Suppress Tablet Mode - if this check box is unselected, each dialog of the application will be opened on the full screen and the TRU keyboard is enabled; if this check box is selected, the TRU keyboard is disabled and each dialog of the application will be opened in the reduced view.

Receiver Settings

The Receiver tab allows to configure a way to display the receiver position in the Position, Position In Time, Base, and Auto Seed dialogs. You can select either Geodetic (Latitude, Longitude, Altitude) or Cartesian (X,Y,Z) coordinate system.

Here you can also specify whether or not to display a message about expired options at start-up.

Solution Colors

In the Status tab, you can view and modify colors used for representing a solution type in the Status dialogs.

Modems Detection

For the Base license type only the automatic modem detecting mode is set on the Modems tab.
