Main Window

The main application window consists of a menu bar, tools area, and status bar.

The menu bar includes the following items:

• Device – contains options, which allow to connect to a device, change general-purpose parameters, and the application mode.
• View – shows or hides the status bar.
• Help – allows you to register the version of the Application, view the Help and the Application version.

The tools area contains icons for the tools included in the current application mode. Each tool is shown by its icon. An icon can also open the next menu level.

The tools area is inactive until a connection to a device is established (the connection icon in the status bar becomes green).

The status bar shows the current receiver mode, and the connection status (the green icon indicates that the device is connected to the application; the gray indicates that there is no connection). You can hide the status bar by unchecking the Status Bar option under the View menu in the menu bar.
