▸AVI
  • avi — Opens AVI web UI. avi "equipName"
  • avilogs — Tunnel via PTX and open WinSCP to AVI logs. avilogs "equipName"
▸AHT / Flight Recorder
  • atmon — Tunnel to FR (port 50000) + launch ATMonitor. atmon "equipName"
  • frdownload — Open FR in WinSCP; for Dash 5 use SSL. frdownload "equipName", frdownload -ssl "equipName"
  • frlogin — Remote Desktop to FR. frlogin "equipName"
  • frreset — Restart FR (multi-vehicle ok). frreset "equipName" [...]
▸PTX
  • vnc — VNC to Screen. vnc "equipName"
  • putty — PTX shell in PuTTY. putty "equipName"
  • cache — Open PTX cache in WinSCP. cache "equipName"
  • delcache — Delete cache + reboot PTX. delcache "equipName"
  • ptxr — Reboot PTX. ptxr "equipName"
  • win — Open Screen home dir. win "equipName"
  • ptxlogs — Open PTX logs in WinSCP. ptxlogs "equipName"
▸GNSS
  • tru — Tunnels to both GNSS receivers + open TRU. tru "equipName"
  • gnsslogs — Proxy to GNSS1/2 and pull logs. gnsslogs "equipName"
▸Misc / Legacy
  • import — Legacy FR ≤3.7 equipment list sync. import
Tip: Check "Not Entered AT" in Equipment Status before reboots.
