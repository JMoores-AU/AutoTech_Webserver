General Information

The General tab shows the modem model, and the product identification information.

Click the Terminal icon to open the Terminal dialog, used to pass commands to the modem directly.

When you click the Cell Info button, the Application detects the cell modem, and, if it finds the modem, connects to it and opens the Settings property sheet for the cell modem.

If a cell module is absent in currently connected modem, the Cell Info button is disabled.
