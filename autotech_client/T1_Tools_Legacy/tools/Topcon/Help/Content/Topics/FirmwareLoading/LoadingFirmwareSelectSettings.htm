Selecting Settings

The Select Setting dialog box allows you to configure the communication parameters between the PC/ Controller and the connected device to load the firmware :

  • Data Packet Size - select the size of the data packet of DTP;
  • Serial Port Settings of Primary Receiver - allows to select a baud rate from the drop down list, when the serial port is used for communication between PC/Controller and the receiver.
  • RTS/CTS check box - enables/disables hardware handshaking for the port.
  • Serial Port Settings of Secondary Receiver - allows to select a baud rate from the drop down list for communication between the primary and secondary receivers. This check box will be displayed, when the secondary board of the MR-2/MM-2 receiver is selected to load the firmware.

Click Next to select the corresponding firmware file(s) for the selected target.
