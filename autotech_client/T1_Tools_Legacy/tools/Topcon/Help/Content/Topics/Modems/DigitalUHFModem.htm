Digital UHF Modem

This section describes specific parameters and functions for the Topcon Digital UHF modem.

There is more powerful version of Digital UHF modem called TRL-35. It is identical to the Digital UHF, but supports different power values (up to 32 watts). TRL-35 has no cell modem.

General Information

The General tab shows the modem model, and the product identification information.

Cell Modem – click the button to view information on the cell module.

Modem Settings

The Settings tab allows setting values for parameters of the connected modem.

The Settings tab displays general communication parameters, and parameters specific for the modem. To change any editable parameter, click on the Value cell of the highlighted parameter and select a desired value from the drop-down list appeared.

• General parameters:
  • Mode – selects Radio or Cell mode to use for communication (the UHF or GSM/CDMA band, respectively).
  • Baud rate, baud – selects a baud rate for the modem’s serial port.
  • Flow Control – enables hardware flow control.
• Radio parameters:
  • Frequency, MHz – assigns an operating channel to the radio modem. Select the desired frequency from the list.
  • Protocol – sets the protocol for data receiving/transmitting.
  • Modulation – selects a modulation type for the base radio modem.
  • Link rate, baud – data transfer speed between the base and rover. This parameter depends on the modulation and the space value.

  • Network has Repeater – enables retransmission in the wireless cluster only for Simplex Transmitter protocol.
  • Power, mW – sets the transmission power for the base radio.
  • Scrambler – when enabled, the data is encrypted to protect against unauthorized access to transmitted information.
  • FEC (Forward Error Correction) – when enabled, activates the operation to control errors in data transmission over unreliable or noisy communication channels.

Modem Functions

The Functions tab allows invoking modem's functions.

The following functions are available for executing in this modem:

• BootConfiguration – the command is used to reboot the modem using selected user settings. Two options are available: to use the default user settings defined by dealer or to use the settings defined by the end-user.
• UploadCCXFile – used to upload the factory and dealer configurations along with the user defined radio settings. After uploading the CCX file, the uploaded file takes into effect after the configuration reboot (see the previous function BootConfiguration).
