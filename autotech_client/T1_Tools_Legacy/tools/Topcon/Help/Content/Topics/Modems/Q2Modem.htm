Q2 Modem

This section describes specific parameters and functions for the Q2 modem.

General Information

The General tab shows the modem manufacturer, model and identification information.

– click the button to open the Terminal dialog, which allows you to send commands to the modem manually.

Cell Modem – allows to view information on the cell module and SIM card inserted.

Modem Settings

The Settings tab allows viewing and changing general communication parameters and parameters specific for the modem. To change any editable parameter, click on the Value field of the highlighted parameter and select a desired value from the drop-down list appeared.

• General parameters:
  • Baud rate, baud – selects a baud rate for the modem’s serial port connected internally with the receiver's serial port.
• Radio parameters:
  • Band – indicates the frequency band and spacing.
  • RX Frequency, MHz – the operating frequency used for data reception.
  • TX Frequency, MHz – the operating frequency used for data transmission.
  • Protocol – sets the protocol for data receiving/transmitting.
  • Modulation – selects a modulation type for the base radio modem.
  • Link rate, baud – specifies the data transfer speed between the base and rover. This parameter depends on the modulation type and the space value.
 
  • Scrambler – when enabled, the data is encrypted to protect against unauthorized access to transmitted information.
  • FEC (Forward Error Correction) – when enabled, activates the operation to control errors in data transmission over unreliable or noisy communication channels.
  • CRC Checking – indicates the checksum use (CRC="Cyclic Redundancy Check") when receiving a package. If the checksum is wrong (the package was received with errors), the data is not applied.
  • Power, mW – sets the appropriate level of output power for the base radio.
  • FCS – allows enabling Free Channel Scan mode.
    In free channel scan mode, the Base radio modem will regularly scan the current operating frequency to determine the degree of a radio noise (interference). Should the level of noise on the current operating frequency be greater than the threshold specified in the Signal Threshold field, the transmitter will stop broadcasting and switch to the next frequency.
• Satel parameters (applied to Satel-compatible protocols):
  • Satel Mode – the modem operation mode.
  • Satel FEC – Forward Error Correction.
  • Satel CRC Checking – indicates the checksum use.
  • Satel TX Addressing – specifies address transmission, (On or Off).
  • Satel TX Address, hex – configures primary address transmission.
  • Satel RX Addressing – specifies address receiving, (On or Off).
  • Satel RX Address, hex – configures primary address receiving.
• GSM parameters:
  • Mode – specifies the GSM modem operation mode. The parameter takes the following values: Off (default), Slave, Master, or Direct.
  • PIN – specifies the PIN code for access to the GSM modem. It can be a 4-digit number.
  • Dial – specifies the phone number that the GSM modem in Master Cell mode will dial in to the remote Slave modem. The number should consist of up to 14 digits.
  • Send Time, s – specifies a timeout interval from 0 to 255 in seconds, used to ensure on reliable communication between the pair of modems (Master - Slave) and avoid unnecessary modem reinitialization. The transmitting modem sends the service word to the receiving modem in every second of the Send Time interval. Note, that the service word will not affect the differential corrections (RTCM or CMR messages) in any way. If the parameter is set to zero, the service word will not be used in data transmission.

Modem Functions

The Functions tab allows invoking modem's functions.

The following functions are available for executing in this modem:

• AddChannel – command allows to add an operating frequency from a list defined by the dealer to the list defined by the user.
• DelChannel – allows to delete an operating frequency from the user defined list.
• FactoryReset – resets all the parameters to their default values.
• GetChannelFCS – shows if a channel can be used in the FCS mode.
• SetChannelFCS – allows to use a channel in the FCS mode.
• SetBand – allows to change the current band (and the step).

Supported bands depend on the hardware type.
