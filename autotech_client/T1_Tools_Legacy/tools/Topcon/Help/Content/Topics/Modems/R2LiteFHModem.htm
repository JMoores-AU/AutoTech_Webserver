R2 Lite FH Modem

This section describes specific parameters and functions for the R2 Lite FH modem.

General Information

The General tab shows the manufacturer, the modem model, and the product identification information.

Modem Settings

The Settings tab allows setting values for parameters of the connected modem.

The Settings tab displays general communication parameters, and parameters specific for the modem. To change any editable parameter, click the Value cell of the highlighted parameter and select a desired value from the drop-down list appeared.

• General parameters:
  • Baud rate, baud – specifies the baud rate of the modem's serial port connected internally with the receiver's serial port. The parameter takes the following values: 19200, 38400 (default), or 57600.
  • Flow Control – enables/disables the RTS line of the modem's serial port connected internally with the receiver's serial port. The parameter takes the following values: RTS/CTS (default) or None.
• Radio parameters:
  • Protocol – sets an operation protocol. The parameter takes the following values: FH915 (default) or FH915 Plus.
  • Location – adjusts the frequency range and RF power level depending on the country. The parameter takes the following values: USA/Canada (default), Australia or New Zealand.
  • Operation Mode – specifies the mode in which the radio modem is functioning. The parameter takes the following values: Receiver (default), Transmitter, Repeater or None (power of the modem is ON, but the modem does not work).
  • Power, mW – sets the transmission power for the base radio modem. The parameter takes the following values: 250 or 1000 mW (default).
  • Link Rate, baud – specifies the rate at which data is transmitted over the RF link using FH915 protocol. The parameter takes the following values: 9600 (default), 12000, 17000, 24000, or 51000 baud.
  • Channel – allows to set up to ten simultaneously transmitting radio modems, without interference, at the jobsite (default value is 1).
  • H/W Version – displays the hardware version of the board.
• Advanced parameters:
  • Net ID – sets a network ID, used to prevent the reception of the messages coming from an external modem's network. The radio modems operating in the same network must have the same Network ID, since a radio link can only be established between modems with the same Net ID. The parameter takes the values in the range from 0 (default) to 255.
  • Rep Net ID – sets a new Net ID parameter for modem working as a repeater. A modem with repeater type carries out retransmission with Rep Net ID that is different from Net ID of receiving network. In this mode, it is possible to set different Net IDs for received and retransmitted signals. The parameter takes the values in the range from 0 (default) to 255.
  • Hot Lock, s – sets time to keep synchronization after signal drop. The parameter takes the following values: 5, 10, 20, 30 (default), or 40 seconds.
  • Save Power – enables/disables power saving mode. The transmitter and receiver must be set to the same mode. The power saving mode is disabled by default.

Modem Functions

The Functions tab allows invoking modem's functions.

The following functions are available for executing in this modem:

• FactoryReset – resets all the parameters to their default values.
