Welcome to the Receiver Utility

The Application is a software product designed for users who need to configure the receiver hardware, or peripheral devices (internal, and external modems, Bluetooth boards, etc.). It is available for installation on desktop computers and hand-held controllers (Mobile devices).

This Help describes version 3.3 of the Application with the Base license type of register intended for general users.

To display the Help on a hand-held controller with Windows Mobile, click the question mark in the upper-right corner of each dialog.

To display the Help on a hand-held controller with Windows CE, click the question mark in the menu bar of Application.

To display the Help on a desktop computer, click the F1 key.
