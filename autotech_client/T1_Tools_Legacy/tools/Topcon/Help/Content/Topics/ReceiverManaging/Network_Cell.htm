Cell

The Cell tab allows you to view the status of the cell modem, configure the cell modem and start the modem to establish connection with GSM Network or another cell modem.

Before starting work with the cell modem make sure that the modem is turned on and SIM card is inserted to the SIM card slot.

The Virtual Communication Devices table contains the following columns:

  • Slot for driver - a slot for uploading a modem driver to a physical modem of the GNSS receiver. There are 4 slots: A,B,C,D. The slot may be the following state:
    • - a modem driver is not uploaded to the slot;
    • - a modem driver is uploaded to the slot. The profile is not started;
    • - a modem driver is uploaded to the slot. The profile is started (The user selects the "Start Current Profile")
    • - a modem driver is uploaded to the slot. The profile is stopped (The user selects the "Stop Current Profile")
  • Hardware - displays the cell modem type for the slot
  • Current State - displays the current status of the slot:
    • Off - the profile of the driver is not started;
    • Running - the profile of the driver is running;
    • Stopping - the profile of the driver is stopping.
  • Report - displays connection status steps of the cell modem, when a profile is running or stopping. There are different status steps for the different drivers, modems and SIM card configuration. Below you can see the steps of connection of GR-5 receiver with Motorola G24 cell modem:
    • The Internet profile is running:
      • M105#Looking for modem - verifying the presence and type of the cell modem;
      • M106#Configuring modem - sending the user parameters for modem initialization;
      • M107#Checking SIM card - verifying the presence and type of the SIM card;
      • M101#Registering - GSM/CDMA network registration in progress;
      • M201#PPP handshake - a step for initiative communication;
      • M205# PPP obtaining IP adress - the cell modem obtains an IP address in order to access internet.
      • M204#PPP connected - the internet connection is established.
    • The Internet profile is stopping:
      • M202# PPP shutdown - a step for disconnection;
      • M203#PPP disconnected - the cell modem is disconnected from GSM/CDMA network
      • M106#Configuring modem - sending the user parameters for deinitialization.
    • The c-master profile is running:
      • M105#Looking for modem - verifying the presence and type of the cell modem;
      • M106#Configuring modem - sending the user parameters for modem initialization;
      • M101#Registering - GSM/CDMA network registration in progress;
      • M102#Dialing - dialing of the slave modem number;
      • M121#Connected - the connection is established.
    • The c-master profile is stopping:
      • M106#Configuring modem - sending the user parameters for modem initialization;
    • The c-slave profile is running:
      • M105#Looking for modem - verifying the presence and type of the cell modem;
      • M106#Configuring modem - sending the user parameters for modem initialization;
      • M101#Registering - GSM/CDMA network registration in progress;
      • M103#Awaiting call - awaiting call from the master modem;
      • M121#Connected - the connection is established.
    • The c-slave profile is stopping:
      • M106#Configuring modem - sending the user parameters for modem initialization.
  • Current Profile - the name of the selected profile.
  • Current Script - the name of script, which is running. A script name is shown if a selected profile is running or stopping. When a profile is not started, the column is empty.

    • Right-click (PC) or tap (Mobile) on the slot with the modem driver to bring up the following controls:

    • Properties - opens the Slot <'N'>: Properties dialog;
    • Start Current Profile - starts the selected current profile. This command is enabled, if a current profile is selected;
    • Stop Current Profile - stops the selected current profile. This command will be enabled, if a current profile is running
    • Select Current Profile - selects one profile from the list. You cannot select a profile, if another profile is running or stopping.

The Parameters panel displays the communications parameters for the selected slot. Right-click (PC) or tap (Mobile) anywhere in the Parameters panel to bring up the controls:

  • Edit - change the communication parameters. The list of editable parameters depends on the selected profile.
    • For the Internet profile you can edit the following parameters:
      • PIN for the SIM card of the GNSS rover receiver modem;
      • APN (Access Point Name) of GSM/CDMA network;
      • Login and Password for GSM/CDMA network;
      • Dial Number of GSM/CDMA network.
    • For the c-master profile you can edit the following parameters:
      • PIN for the SIM card of the GNSS receiver modem;
      • Dial Number of the GNSS receiver modem.
    • For the c-slave profile you can edit the PIN for the SIM card of the GNSS receiver modem.
  • View – select how you want to view the options, in detail (Detailed) or quick view (Overview). In Overview mode, the Application will hide Id and Name columns.
