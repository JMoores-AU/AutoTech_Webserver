Wi-Fi

The Wi-Fi tab allows you to configure and manage the Wi-Fi module in your receiver.

• Mode: Selects the operating mode for the Wi-Fi module.
  • Access Point – the Wi-Fi module will act as an access point (AP).
  • Client – the Wi-Fi module will act as a client.
  • Off – the Wi-Fi functionality is disabled.
• MAC - specifies your receiver's unique hardware number on a network.
• The Access Point area contains the settings used for configuring the AP.
  • SSID – specifies the name of the wireless network. Wireless clients must use this name to associate with the AP.
  • Password – enters a password. A wireless client must enter the password to authenticate itself to the AP. With an empty password, a client will authenticate to the AP without entering a password.
  • IP Address – specifies the IPv4 address of the AP.
  • Mask – specifies the network mask of the AP.
  • Don't broadcast SSID check box – When checked, disables the AP to broadcast the SSID specified in the SSID control. When an AP broadcasts the SSID, wireless clients without a specified SSID can discover and associate to the AP.
• The Client area contains the settings used for configuring the Wi-Fi module in Client mode.
  • SSID – specifies the name of the wireless network. As a wireless client, you use this name to associate with the AP.
  • Password – enters a password. As a wireless client, you must enter the password to authenticate to the AP.
  • IP Address – shows the IP address of a client assigned by the WLAN.
