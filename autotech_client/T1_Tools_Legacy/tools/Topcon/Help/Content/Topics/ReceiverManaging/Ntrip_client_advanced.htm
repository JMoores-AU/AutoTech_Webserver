Advanced Settings

The Advanced Settings dialog allows you to specify the following extra parameters of the rover receiver:

• Ntrip Client's Port — specifies Ntrip port of the rover receiver.
• Reset RTK engine if a new Ntrip sessions starts check box — when checked, after sending the Reset RTK command to the receiver, a next Ntrip session will be started.

The Send NMEA GGA to Caster area contains the settings used to configure the GGA sentence sending .

• Do not send — if it is selected, the rover receiver will not send GGA to the NtripCaster.
• Once after connecting — if it is selected, the rover receiver will send GGA one time after connecting with the NtripCaster.
• Period, s — if it is selected, the rover receiver will send GGA with the specified period.
