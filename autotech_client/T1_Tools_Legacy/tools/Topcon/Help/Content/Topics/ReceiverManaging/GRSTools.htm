GRS Tools

When running on GRS-1 GNSS receiver, the Device drop-down menu contains the GRS Tools item.

The GRS Tools item opens the GRS Tools dialog that allows to manage the receiver, modem and compass.

The dialog includes the following tabs:

  • Receiver – allows to control power parameters of the internal GNSS receiver.
  • Modem – allows to control power parameters of the internal modem.
  • Compass – allows to display the current North/South direction, the angle to North, the tilt of the unit, and also to calibrate the compass.
