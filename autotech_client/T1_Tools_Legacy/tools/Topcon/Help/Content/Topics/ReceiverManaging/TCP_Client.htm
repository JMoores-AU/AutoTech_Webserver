TCP Client

The TCP Client tab allows you to configure the receiver as the TCP Client.

  • Check box Enable TCP Client - select the checkbox to activate the TCP Client.

In the Remote TCP Server panel you can specify the following settings for the TCP Client:

  • IP Address - enter an IP address of the remote TCP server which the TCP client will connect to.
  • Port- enter an number of TCP port of the remote TCP server which the TCP client will connect to.
  • Timeout, s - specify the amount of time, in seconds, the TCP client will try to re-establish connection to the TCP server before terminating the connection.
