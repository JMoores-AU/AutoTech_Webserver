ase Station Parameters

The Base tab allows configuring a reference base station and the Auto Seed functionality.

• Antenna Type – Antenna type descriptor for RTCM 2.3 and 3.0. The Antenna Type combo box is populated with values, which are read from the receiver antennae database.
• Base Station Coordinates – This parameter specifies coordinates of the Antenna Reference Point or Phase Center L1. Click to open a dialog, where you can specify coordinates.You can enter them in Geodetic or Cartesian representation depending on which coordinate system you select in theSetup window on the General tab.

Coordinates of ARP and Phase Center L1 are interrelated. When a valid Antenna Type is selected, secondary (not selected) coordinates are calculated based on information about the selected antenna from the receiver database.
When no antenna is selected, or the receiver database does not contain information about the antenna, secondary coordinates are set equal to primary (selected) ones.
