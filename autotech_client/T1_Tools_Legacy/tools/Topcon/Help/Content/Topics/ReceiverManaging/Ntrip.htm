Ntrip

The Ntrip icon opens the Ntrip dialog where you can configure Ntrip functionality. The content of the dialog depends on the connected receiver.

Topcon HiPer II, HiPer V, HiPer SR, GR-5 and Net-G3A receivers have the following Ntrip functionality:

• one NtripClient and one NtripServer.

For these receivers the Ntrip dialog contains two tabs: Client and Server.

Topcon HiPer HR and Net-G5 receivers support full range of NTRIP functionality including:

• one NtripClient and one NtripServer (Server 1 );
• two extra NtripServers: Server 2 and Server 3;
• NtripCaster.

For these receivers the Ntrip dialog contains the followings tabs: Client, Server 1, Server 2, Server 3 and Caster.

Topcon HiPer VR receiver supports the following range of NTRIP functionality including:

• one NtripClient and one NtripServer (Server 1 );
• two extra NtripServers: Server 2 and Server 3;

For this receiver the Ntrip dialog contains the followings tabs: Client, Server 1, Server 2, Server 3.
