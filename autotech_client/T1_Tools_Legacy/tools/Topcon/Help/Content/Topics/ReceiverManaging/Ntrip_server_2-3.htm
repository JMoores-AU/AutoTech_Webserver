Server 2 / Server 3

The Server 2 / Server 3 tab allows you to configure connection of the Net-G5, HiPer VR / HiPer HR receivers to an external NtripCaster :

• Enable check box – when checked, the NtripServer functionality for the receiver is enabled. Activation of the function for this Server does not change the NtripClient state for the receiver.
• Caster – specifies the DNS host name or IP address of an NtripCaster to which the receiver, acting as an NtripServer, will connect. Click to open the Caster dialog.
• Mountpoint Name – specifies an NtripCaster mountpoint where the NtripServer will transmit the correction data.
• Protocol – defines the way of receiving correction data. You can select the following from the drop-down list: Ntrip v1.0 , Ntrip v2.0 with HTTP or Automatic Selection.
• Output – specifies the type of the correction data to send out from the NtripServer to the selected NtripCaster mountpoint. Click to open the Output dialog.
• Current State – shows the current state of the parameter for Ntrip port A (Server 1), Ntrip port B (Server 2) or Ntrip port C (Server 3).
• Last Error – reports the errors occurred while using Ntrip.

Click the User button to open the User Credentials dialog, where you can specify the user name and password required by Ntrip.

Click the Proxy button to open the Proxy Server dialog, where you can specify the IP address and TCP port number of a proxy server located between the NtripServer and NtripCaster.

Click the Advanced button to open the Advanced Settings dialog, where you can specify extra parameters for the base receiver.
