Log <current file name>

The Log <current file name > dialog allows you to view and configure some individual settings of the current log file.

• Elevation Mask, ° – specifies the elevation mask threshold value for the TPS messages to be output to the file.
• Prefix – specifies the file name prefix used as a part of the name of the new file that the receiver will create.
• Logging Period, s – specifies the interval to be used for data logging. It can be set to the values from 0 to 86400 seconds; the default is 1 second.
• Rotation Period, s – specifies the duration time of each log file created in AFRM.
• Message Set for HiPer VR , HiPer HR and Net-G5 receivers – allows you to select any message set from five messages sets defined in GRIL. More about the messages sets.
• RINEX Version for HiPer HR and Net-G5 receivers – allows you to select the desired RINEX version from the list. If you select "None" in this list, the RINEX conversion will not be enabled.

The conversion of the log file to the RINEX file format is started immediately after the log file is closed.
• Use for MINTER & AFRM check box – when selected, the specified log parameters will be used after pressing the MINTER FN key and starting of AFRM.
