Casters

The Casters tab allows you to configure internal NTRIPCasters.

• Enable check box — when checked, the NtripCaster functionality for the base receiver is enabled.
• Caster Name: — specifies the common name for all three casters.
• Country: — specifies a country name from the drop-down list. This name is used as Caster country in source table entry. This name is the common name for all three casters.
• Authentication: — specifies authentication parameters. The selected parameter will be used for all three casters:
  • None — an access protection does not apply,
  • Basic — basic way is used for access protection,
  • Digest — digest way is used for access protection.

Click the Caster:1,2,3... button to open Ntrip Casters dialog, where you can specify the NtripCaster's parameters.

Click the Users button to open Ntrip Casters Users dialog, where you can set up users for the NtripCaster connection.
