FTP

The FTP dialog allows configuring the internal FTP server of the Topcon receiver:

• FTP Port – specifies the TCP port the receiver is listening for FTP connections. Values: 1…65535. Default value: 21.
• Timeout, s – specifies a period of time that must elapse before an inactive connection will be terminated. Values: 1…231-1 seconds. Default value: 600 seconds.
• Login – specifies the user name required by the internal FTP server of the GNSS receiver for an external client to connect. The user name is case sensitive and can contain up to 31 alphanumeric characters. The default is topcon. This setting is only available for the MM2/MR2, HiPer VR, HiPer HR and Net-G5 receivers.
• Password – specifies the password required by the internal FTP server of the GNSS receiver for an external client to connect. The password is case sensitive and can contain up to 31 alphanumeric characters. The default is topcon. This setting is only available for the MM2/MR2, HiPer VR, HiPer HR and Net-G5receivers.

Note: Depending on the receiver and OEM boards, you can establish different number of the simultaneous connections to the internal FTP server:

• MM2/MR2 (Single or Dual) receivers and B125 OEM board support up to ten simultaneous connections to the FTP server;
• NET-G5, GNR5, HiPer HR, HiPer VR, GRX3 receivers and B310 OEM board support up to ten simultaneous connections to the FTP server;
• NET-G3A receiver supports up to five simultaneous connections to the FTP server;
• all other receivers and B210 OEM board support only one connections to the FTP server.

Note: Any values changed in this dialog will be stored in the permanent area of NVRAM. It means they will be left intact after you clear the NVRAM or reset all parameters to default.
