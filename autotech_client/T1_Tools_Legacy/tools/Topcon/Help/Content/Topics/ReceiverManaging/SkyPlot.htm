Sky Plot

The Sky Plot tab displays the graphic position of satellites on the sky. You can also enable Planning Mode to view available satellites with their trajectories ± 12 hours from the current time.

Each satellite system is marked in a certain color:

• GPS in green ()
• GLONASS in yellow-orange ()
• Galileo in cyan ()
• BeiDou in magenta ()
• SBAS in purple ()
• QZSS in blue ()

The ‘wings’ on the satellite icon designate that the satellite is used in position computation. The absence of ‘wings’ on the satellite icon means that the satellite is not used in the computation for some reason (e.g. it is below the elevation mask).

The PRN or SN (slot number for GLONASS) of each satellite is displayed under satellite icons.

Red circle designates the elevation mask.

Clicking on a satellite icon displays a tool tip that shows the information about the satellite.

Selecting Satellite Systems for Displaying on the Sky Plot

You can specify which satellite system to show/hide on the sky plot.

1. Right-click (PC version) or tap (Mobile version) anywhere in the Sky Plot tab.
2. Check the satellite systems you want to view. The sky plot only displays the satellite systems that are supported by the connected receiver.
3. Uncheck the satellite systems you want to hide from the sky plot.

Viewing or Hiding Satellite Trajectories on the Sky Plot

To show the satellite trajectories across the sky, right-click (PC version) or tap (Mobile version) anywhere in the Sky Plot tab and check Show Orbits.

To hide the satellite trajectories, uncheck Show Orbits.

Viewing Satellites in Planning Mode

You can use the sky plot in Planning mode to view available satellites as they cross the sky ± 12 hours from the current time. In Planning mode, the Application uses the latest GNSS almanacs available in the receiver to show the satellites. If an almanac is unavailable for the corresponding satellite system, its satellites will not be shown on the sky plot.

To view satellites after the current time, slide the indicator along the time scale to the right or click on the right arrow to set the desired time.

To view satellites before the current time, slide the indicator along the time scale to the left or click on the left arrow to set the desired time.

In the upper right corner, the Time field shows the indicator's offset from the current time.

When you toggle between standard and planning modes, positions of satellites on the sky plot may change slightly. This is not a symptom of faulty operation; the Application is functioning as designed.

You may see these slight position changes because orbital data, used to show satellite trajectories, come from sources of different accuracy. In standard mode, the Application uses accurate orbital data from ephemeris, while in planning mode, it uses coarse orbital data from almanac.
