Caster

The Caster dialog allows you to specify the IP Address or Domain Name and the TCP Port Number of :

• an NtripCaster to which the rover receiver, acting as NrtipClient, will connect,

or

• an NtripCaster to which the base receiver, acting as NrtipServert, will connect.
