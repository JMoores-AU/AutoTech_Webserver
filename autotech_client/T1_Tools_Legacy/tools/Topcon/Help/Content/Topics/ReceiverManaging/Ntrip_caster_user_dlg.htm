Ntrip Caster User

In the Ntrip Caster User dialog you can add a new user or edit parameters of the existing user which will be have an access to one NtripServer:

• Name — specifies a user name.
• Password — specifies a password for the user.
• Access — specifies a NtripCaster where the user will have access. If "No Access" is set, the user cannot connect with any NtripCaster.
• Valid From — specifies a starting data and time of the access validity period for the user.
• Valid Untill — specifies a finishing data and time of the access validity period for the user.
