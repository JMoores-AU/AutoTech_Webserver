Event A / Event B

Topcon receivers are capable of measuring and recording event times in the specified reference time system with high accuracy. It is possible to measure the time of either the rising edge or falling edge of the input event signal. Most of the TPS receivers can accommodate up to two external event pins, a (Event A) and b (Event B).

In the Event A and Event B tabs you can configure receiver to record the time of any input pulse (with the input low voltage levels from 0.0 V to +0.8 V and the input high voltage levels from +2.0 V to +3.3 V).

The tabs contains the following parameters:

  • Enable check box — enables the receiver to measure and log event times, in the specified reference time.
  • Edge — specifies whether the receiver will measure the time of the rising edge or falling edge of the input event signal.
  • Ref.Time — specifies the reference time that the event reception time will be synchronized with; either GPS system time, GLONASS system time, UTCus (UTC USNO) system time, or UTCsu (UTC SU) system time.
  • Tied with Ref. Time check box — when selected, the receiver will synchronize the event signal with the chosen reference time. Otherwise, event signal will be synchronized with the receiver's internal clock or with an external reference frequency applied to the receiver.
  • Clock Synchronization check box — when selected, the receiver's internal clock is synchronized with the corresponding edge of the event signal arrived.
    • Status — indicates whether the receiver clock is actually synchronized with the event signals. “On” means that the synchronization has been done successfully.
    • Repeat Synchronization button — clicking on the button performs a synchronization with the event signal.
