Advanced Settings

The MINTER dialog allows you to edit the following advanced settings:

  • FN Key Mode / PWR Key Mode – allows to program the receiver's behavior after pressing the FN or PWR key.

    • PWR Key Mode is displayed for those Topcon GNSS receivers (HiPer SR,HiPer VR HiPer HR, GCX2, GCX3) where the power button performs multiple functions.
    • Led blink mode switch – press and hold down the FN or PWR key for less than a second to switch the receiver between standard and extended information modes. Refer to the receiver's Operator's Manual for details about these modes. This is default value.
    • Occupation mode switch – press and hold down the FN or PWR key for less than a second, switches the receiver between static and kinematic survey modes when recording a file.

    For the HiPer SR receiver, these settings are grayed out and inactive.
  • Initial Data Collection Dynamic Mode – allows to specify the initial dynamic mode for all the new files opened through-MINTER when FN Key Mode or PWR Key Mode is set to the Occupation mode switch mode. Default value: Static survey mode.
  • Data Recording Auto-start – allows to program the receiver's behavior in the event of a power failure. The default value is Off.
    • Off – receiver will not resume data logging on start-up.
    • On – should a power failure occur when the receiver is logging data, the receiver will automatically open a new file and resume data recording when power has been returned. From a functional point of view, this is equivalent to pressing the FN or PWR key once the receiver is powered on again.
    • Always – the autostart mechanism is launched at the receiver start time, irrespective of whether the power failure occurred while data recording or not.
  • Always Append to the File – allows to specify the name of an existing file to append new data to. The file name can be up to 20 characters long.
