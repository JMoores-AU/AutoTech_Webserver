Advanced RTK Parameters

The Adv. RTK dialog is designed to customize the RTK engine.

  • VRS Mode – Enables or disables the receiver VRS mode:
    • Enabled (checked) – RTK will assume that the reference station in use is Virtual Reference Station (VRS).
    • Disabled (unchecked) – RTK will assume that the reference station is the real one.
  • IF Baseline, m – The threshold value for applying iono-free combinations for RTK solution. If the baseline length is greater than or equal to the specified value, the RTK engine starts to use the dual frequency code and carrier phase measurements to form iono-free combinations in order to eliminate ionospheric error. For the different versions of firmware the following values are used:
    • 4000 meters for firmware version up to 3.5;
    • 20000 meters for firmware version 3.6;
    • 10000 meters for firmware version 4.0 and higher.
  • Weighting Based On Residuals – Controls applying weights to measurements based on VRS residuals (RTCM 3.1 MT1030, MT1031). Default: Disabled.
  • Maximum Residual Value, mm - Threshold in [mm] for rejecting measurements based on value of VRS residual (RTCM 3.1 MT1030, MT1031). "0" means no rejection. Default: 0.
