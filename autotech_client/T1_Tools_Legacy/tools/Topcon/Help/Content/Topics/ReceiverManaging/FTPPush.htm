FTP Push

The FTP Push dialog allows configuring an FTP client in your receiver. With the FTP client, the receiver can automatically upload files to an external FTP server.

• Enable File Transfer – the checkbox enables file transfer. After you configure the FTP server settings and select this option, the FTP client starts transferring files to the FTP server. The FTP client will only transfer current files after they are closed, not every file is stored in the memory.
• Folder – specifies the path to the directory on the FTP server where the receiver will transfer files. The directory path can contain up to 127 alphanumeric characters where path components are separated by the slash (“/”).
• Click the icon next to the Folder setting to specify an IP address and credentials of the FTP server.
• Error – shows you any errors encountered during an FTP session.

Note: Any values changed in this dialog will be stored in the permanent area of NVRAM. It means they will be left intact after you clear the NVRAM or reset all parameters to default.
