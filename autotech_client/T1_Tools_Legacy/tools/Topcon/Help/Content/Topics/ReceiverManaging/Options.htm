Options

Click the Options icon to manage receiver options. The Receiver Options dialog box shows you information about loaded options and uploads OAF files.

The options are organized into eight groups:

  • Tracking – controls tracking of satellite systems and their signals, output of raw measurements and activation of various tracking algorithms.
  • Positioning – controls various positioning modes, including DGNSS, PPP, RTK, HD2, and other.
  • Interfaces – controls communication interfaces, memory, and timing signals.
  • Input/Output – controls input and output of DGNSS/RTK data in different formats and the operation of frequency source.
  • Machine Control – controls mmGPS.
  • Agriculture – controls the HiPer AG.
  • Region Lock – controls the geofence's regions.
  • Other – miscellaneous options, for example, Cinderella, COCOM, etc.

The following information is displayed for each loaded option:

  • Friendly Name – displays a name for the receiver option in a user-friendly manner.
  • Option ID – displays a name for the receiver option in a GRIL convention. Available in the Detailed view only.
  • Current – indicates the current value of the option.
  • Purchased – indicates the purchased value of the option.
  • Leased – indicates the leased value of the option. Leased options may display incorrectly if the receiver does not have a valid time.
  • Exp. Date – indicates the date the leased option is expired. Available in the Detailed view only.

The options list depends on the firmware version loaded into the receiver.

Right-click (PC) or tap (Mobile) anywhere in the window to bring up the following controls:

  • Refresh – refreshes the options list.
  • Upload OAF – uploads a new OAF file. By default, the Application offers to use the <receiver ID>.tpo file name.
  • Save to File – saves the options list to a text file. By default, the Application sets the <receiver ID>.opt.txt file name. You can save the options in two different views into file :
    • If you select View, the options will be saved into the file as they are displayed the Receiver Options dialog box.
    • If you select Options, the Application saves the options as a reply to the print,/par/opts:on command:

      • RE030%% _GPS={cur=1,purchased=0,leased=1,date=30517},

      RE030%% _GLO={cur=1,purchased=0,leased=1,date=30517},

  • View – select how you want to view the options, in detail (Detailed) or quick view (Overview). In Overview, the Application will hide the Option ID and Exp.Date columns.
