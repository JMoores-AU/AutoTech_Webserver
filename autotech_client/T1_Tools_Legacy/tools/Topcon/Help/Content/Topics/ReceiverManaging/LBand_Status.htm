L Band

The L Band tab displays the list of the L-Band receiver parameters. This list is enabled after the L-Band receiver locks onto the selected satellite.

The tab contains the following parameters:

The L-Band Receiver Status panel:

  • Carrier Frequency, Hz – shows the carrier frequency of the selected satellites;
  • Symbol Rate – shows the nominal symbol rate being used at this moment;
  • Service ID, hex – shows the service ID specified by a user;
  • Scrambler Vector , hex – shows the current value of the scrambler vector;
  • Current Frequency, Hz – shows the nominal value of the current frequency of L-Band receiver;
  • Signal Quality, dB – shows the received power level in dB;
  • Signal to Noise ratio, dB*Hz – shows the current signal to noise ratio, in dB*Hz;
  • Message Age, sec – shows the age of the last message received;
  • Decoded Service, hex – shows the ID of service the L-Band receiver is working with;
  • Active Service – shows the name of service the L-Band receiver is connected to;
  • Bit Error Rate, % – shows the ratio of the corrupted bits number to the received bits number, in percentage terms;
  • Tracking Status – shows the signal tracking status of the L-Band receiver.

The Status of Services panel:

  • Service – shows the name of service the L-Band receiver is connected to;
  • Solution Mode – shows the type of measurements used for the position calculation. For TopNET Global service you can see the following modes:
    • Empty – the PPP engine is not selected;
    • Apex2 – PPP engine uses GPS + GLONASS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GPS and GLONASS, respectively;
    • GPS – PPP engine uses GPS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GPS;
    • GLONASS – PPP engine uses GLONASS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GLONASS;
  • Good Messages – number of correct messages;
  • Bad Messages – number of corrupted messages;
  • Status – shows the list of warnings and errors for the current service;
  • Station ID – shows the type of measurements used for the position calculation.

The Subscriptions panel:

  • Service – shows the name of service the L-Band receiver is connected to;
  • Solution Mode – shows the type of measurements used for the position calculation. For TopNET Global service you can see the following modes:
    • Empty – the PPP engine is not selected;
    • Apex2 – PPP engine uses GPS + GLONASS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GPS and GLONASS, respectively;
    • GPS – PPP engine uses GPS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GPS;
    • GLONASS – PPP engine uses GLONASS measurements with precise coordinates of satellites data and time scale shift of each satellite in regard to time scale of GLONASS;
  • Constellation – shows the type of measurements used for the position calculation.
  • Expiration Date – shows information about expiration date of the subscription.
  • TerraSTAR PAC – shows the service product activation code. You will use this value to obtain a subscription.
  • Engine Version – shows the number of PPP engine firmware version.
