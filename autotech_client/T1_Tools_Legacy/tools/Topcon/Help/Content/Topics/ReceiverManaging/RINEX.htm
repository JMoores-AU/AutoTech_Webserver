RINEX

The RINEX tab configures conversion of binary TPS files to ASCII RINEX files for HiPer HR and Net-G5 receivers.

• Enable – when the check box is selected, the software activates RINEX conversion.
• SNR Observation – when the check box is selected, the software includes signal-to-noise values to RINEX observation files. RINEX observation files include SNR values only if their TPS counterparts have the corresponding binary SNR messages. In a TPS file, SNR values can be stored in the EC, CE, E1, 1E, E2, 2E, E3, 3E, rM, rS, and rD messages.
• Smooth Pseudoranges – when the check box is selected, pseudorange smoothing corrections are added to RINEX observation files. RINEX observation files contain smoothed pseudoranges only if their TPS counterparts have the corresponding binary messages with smoothing corrections. In a TPS file, pseudorange smoothing corrections can be stored in the CC, cc, C1, c1, C2, c2, and C3 messages.
• Hatanaka Compression – when the check box is selected, the RINEX observation files are archived using the Hatanaka compression format.
• Enable V.3 File Naming – when the check box is selected, RINEX v.2 naming convention is used both for version 2 and version 3 RINEX files.
• ZIP Compression – allows to select the compression level for the RINEX files.
  • Off – the receiver will not compress the RINEX files.
  • Fast – the receiver will compress the RINEX files with the fastest compression time at the cost of lower compression level.
  • Maximum – the receiver will compress the RINEX files with the highest level of compression at the cost of a slower compression time.
• Local Folder – allows to specify the folder name which will be created in your receiver, and the path to the destination folder on the local storage medium where you want the receiver to save the RINEX files. You can find the compressed RINEX *.zip file in the <current drive>/<receiver model>/<receiver ID>/<your_local_folder_name>.
• FTP Folder – specifies the path to the destination folder on an external FTP server where you want the receiver to save the RINEX files. Click the icon next to the FTP Folder setting to specify an IP-address and credentials of the FTP server.
• Conversion Status – displays one of the following status of the TPS-file conversion:
  • Off – the RINEX conversion is disabled or a single RINEX conversion is over.
  • Pending – the RINEX converter is waiting for the next conversion.
  • Running – the RINEX conversion is currently running.

Click the Configure Header button to open RINEX Header dialog, where you can specify RINEX header records.

Click OK to save the settings to the receiver.
