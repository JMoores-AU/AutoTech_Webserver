Import Settings From File

The Import Settings to File icon opens the dialog where you can import an external file (*.json) with receiver parameters to the receiver.

To import receiver settings:

1. Click and select the desired file of the receiver parameters to import.
2. Clicking the Next button starts analyzing of the receiver parameters. The parameters will be shown in the Select Parameters To Import dialog. Part of the parameters are ready to import by default ( set for the parameter), and part of the receiver parameters will be not imported to a file (set for the parameter). The parent check box is gray, if one or more child check boxes are not selected:

To find more information about export or import of the receiver parameters, click here.

3. Select parameter(s) for import by checking the corresponding check boxes.
4. Click Next >. The Progress dialog displays the progress of import. If a parameter was imported to the receiver, the Successfully Imported messages is shown for the parameter.
5. Click Finish.

The options list depends on the firmware version loaded into the receiver.

 

6. Parameters are not included in the Import list by default: all ports (Serial, Modem, USB, TCP, Bluetooth, Ntrip, MVP) parameters, IP addresses for Ethernet and Wi-FI, the status (enable of disable) of the TCP Client, NTRIP Server2 (3), Caster, RINEX conversion and Message Sets.

You can select any parameter(s) for import, by checking their check boxes.
