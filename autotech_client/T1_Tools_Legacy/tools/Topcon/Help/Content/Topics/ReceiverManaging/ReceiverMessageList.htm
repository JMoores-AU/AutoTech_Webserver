Receiver Message List

The Receiver Message List dialog consists of upper tree control and lower list control. The tree control displays all the messages supported by the currently connected receiver. The list control displays the messages to be added.

  • To add a highlighted message from the upper tree control to the lower list control you can either double-click it, or press <space> or use the pop-up menu. This message will be added to the end of the message list in the lower list control.
  • To change the order of the messages in the list control, use the Cut /Paste commands of the pop-up menu for the highlighted message. PC version also supports the drag-and-drop method;
  • To delete the messages in the list control, highlight message(s) from the set and click the Delete messages command in the pop-up menu;
  • To delete all messages of the set, click the Delete all messages command in the pop-up menu.
