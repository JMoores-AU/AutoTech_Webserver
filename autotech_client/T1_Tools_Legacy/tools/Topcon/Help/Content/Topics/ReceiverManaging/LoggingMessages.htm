Default Message Set

The def <n> screen allows you to view and edit a list of messages being logged into the file, and also parameters of a selected message. You can add or remove messages or modify their scheduling parameters using the pop-up menu options. The receiver will keep the customized message set in the NVRAM for further usage.

The def, def1, def2, def3 and def4 screens are available for HiPer VR, HiPer HR and Net-G5 receivers.

By default, the def4 screen does not contain any messages. You can use this message set to create your own message set.

Right click in any place of the screen to open pop-up menu. The menu commands allow you to do the following:

  • Add new messages – opens the Receiver Message List dialog to add new message(s) to the end of the message list.
  • Edit schedule – opens the Receiver Message dialog to specify the schedule of the highlighted message and opens the Bulk Schedule Change dialog to specify the schedule of the highlighted messages group.
  • Edit message set – opens the Receiver Message List dialog to:
    • add any message from of upper tree control to the end of the message list in the lower list control.
    • edit message order in the message set using Cut /Paste commands for the highlighted message;
    • edit a message (opens the Receiver Message List dialog for the highlighted message);
    • delete the highlighted message from the set;
    • delete all messages of the set.
  • Delete messages – deletes the highlighted message(s).
  • Delete all messages – deletes all messages of the set.
  • Restore Defaults – restores the list of output messages to the factory defaults.
  • Import – allows to import any messages set to the current messages set. This command is enabled for HiPer VR, HiPer HR and Net-G5 receivers.
  • Refresh – renews the list of output messages with the last made changes.
