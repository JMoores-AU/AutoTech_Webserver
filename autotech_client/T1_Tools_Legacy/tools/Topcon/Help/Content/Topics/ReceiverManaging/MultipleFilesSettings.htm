Multiple Files Settings

Use the Multiple Files Settings dialog to configure naming scheme and duration of multiple files for real-time data logging to the computer or controller.

• Prefix – the beginning part of the name for each file. By default, the Site name is used as a prefix.
• Suffix – the ending part of the name for each file.
  • Incrementing Counter –  the suffix will contain a number varying between 0000 and 231-1.
  • Date & Time – the suffix will contain a date in the format YYMMDDhhmm; where YY is year in the range of 00 to 99, MM is month in the range of 01 to 12, DD is day in the range of 01 to 31, hh is hour in the receiver time scale, and mm are minutes in the receiver time scale.
• Initial Value – enabled when the Incrementing Counter parameter is selected. Here you can specify the first number to use for file counts. This number will increase by one with each file.
• New File Every – specifies how often you want to create a new file. You can select among minute, hour, or day. Note, that files may be shorter than the interval entered in these fields.
• Overwrite Existing Files – over writes existing files that have the same name as new ones.
