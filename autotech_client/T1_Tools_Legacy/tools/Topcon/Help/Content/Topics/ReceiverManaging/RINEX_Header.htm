RINEX Header

The RINEX Header dialog allows you specify the header content of the RINEX file which will be created.

• Country – select the desired country code from the drop-down list.
• Agency – specify a company name.
• Observer – specify a name of the observer.
• Marker Name – specify a name of the antenna marker.
• Marker Number – specify a number of the antenna marker.
• Antenna Number – specify an antenna number.
• Antenna Position – click to open the Cartesian Coordinates dialog, where you can specify the approximate coordinates of the phase center L1 in the Cartesian coordinate system. These coordinates will be used in the RINEX file header.

If you do not specify any coordinates in the fields, the RINEX convertor will use the average values of coordinates from the TPS-file.
