Select Serial Port

For the serial ports and USB devices, this dialog shows physical and friendly names for the ports/devices (if available).

The Application remembers the last used transport, and the last successful port name for every transport, so it is easy to connect to the same device.


If there is a friendly name for the port, you will see both the friendly and physical name. Otherwise, only the physical name is displayed.

To use a port, do one of the following, depending on how you want to select the port/device:

• Double-click (PC) or tap (Mobile) the desired port.

OR

• Highlight the port, right-click (PC) or tap and hold, click/tap Select the port from the pop-up menu.

To view details about a port/device, highlight the port, right-click (PC) or tap and hold, click/tap Device info from the pop-up menu.

To scan for any new Bluetooth devices, right-click (PC) or tap and hold anywhere within the dialog, click/tap Refresh from the pop-up menu.
