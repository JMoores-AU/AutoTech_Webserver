Connecting to the Device in Receiver Managing

To establish a connection to the device, select Device-> Connect from the main menu. The Connection Parameters dialog box displays.

The following connection parameters are available in Receiver Managing mode.

• Connect Using – select either Serial Port, Bluetooth, Bluetooth LEor Network for communication. PC version also supports USB connection.

Bluetooth transport may not be available in the following situations:

  • the device-specific Bluetooth stack is not supported by the Application;
  • the Bluetooth power is turned off (for WM/CE);
  • Bluetooth dongle is not attached (desktop).

If the Bluetooth power is off, turn it on before using the Bluetooth manager.

If the Application edition for the specified platform does not support already Bluetooth stack installed, try to use Microsoft Bluetooth stack, or use virtual serial ports provided by a Bluetooth manager for your Bluetooth device.

Windows Server operating systems do not support Bluetooth.

 

Bluetooth LE transport can be available for the following cases:

  • the operating system of the PC or Controller is Windows 8 or later version;
  • the Bluetooth LE-only or dual-mode (combined Bluetooth LE + Bluetooth Clasic) dongle can be used;
  • the standard Microsoft Bluetooth LE stack, rather than a 3rd party stack is used:

To find more information how to pair with a Bluetooth LE Topcon receiver, click here.

 

To use USB connection, install a Topcon USB driver designed for your specific OS version and system type (32- or 64-bit).

 

• Port name – shows physical and friendly name for the serial port. The Application remembers the last used transport, and the last successful port/device name for every transport, so it is easy to connect to the same port/device.
If there is a friendly name for the port/device, you will see both the friendly and physical name. Otherwise, only the physical name is displayed. Click the Browse button to choose a port from the Select Port dialog and check the port details.
• Device Name – shows the receiver name for the Bluetooth, USB, and network connections. Click the Browse button to select the receiver from the list. For a Bluetooth connection, use this button to search for and add Bluetooth devices. For a network connection, use this button to configure the receiver's network parameters (IP address, TCP port, and password).
• Receiver Port – selects an internal port of the GNSS receiver which is used to connect to an external GNSS receiver (the Secondary Receiver box is checked). Currently, this option is mostly intended for GRS-1 and MR-2 platforms.
• Connect – press to establish a connection.
